Грамматика

Формальное определение грамматики

Грамматика G = (VT, VN, P, S), где:

  • Терминальные символы VT = {a, b, ..., z, A, B, ..., Z, 0, 1, ..., 9, +, -, /, *, {, }, (, ), ;, _, " , =}
  • Нетерминальные символы VN = {<start>, <const>, <val>, <equal>, <id>, <quote>, <string>, <quote_second>}
  • Целевой символ Z = <start>

Определения:

  • symbol ∈ VT
  • digit → '0' | '1' | ... | '9'
  • letter → 'a' | 'b' | ... | 'z' | 'A' | 'B' | ... | 'Z'

Правила вывода P

  1. <start> → 'const ' <const>
  2. <const> → 'val ' <val>
  3. <val> → letter <id>
  4. <id> → letter <id> | digit <id> | '_' <id> | '=' <equal>
  5. <equal> → '"' <quote>
  6. <quote> → symbol <string> | '"' <quote_second>
  7. <string> → symbol <string> | '"' <quote_second>
  8. <quote_second> → ';'
